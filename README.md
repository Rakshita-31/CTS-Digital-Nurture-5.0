## 🚀 How to Run

**Frontend Modules (HTML/CSS/JS/Bootstrap):**
- Open the respective `index.html` in any modern browser
- Recommended: Use **Live Server** extension in VS Code

**SQL Module:**
- Use any MySQL-compatible online compiler:  
  [OneCompiler MySQL](https://onecompiler.com/mysql) or [W3Schools SQL](https://www.w3schools.com/sql/trysql.asp)
- First run `database_setup.sql` to create tables and insert sample data
- Then run individual `exercise_XX.sql` files

**Java Module:**
- Compile: `javac FileName.java`
- Run: `java FileName`
- Or use any online Java compiler: [JDoodle](https://www.jdoodle.com/online-java-compiler-ide/)
- IDE recommended: IntelliJ IDEA / Eclipse / VS Code with Java Extension Pack
